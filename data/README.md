# ML Metrics and Results Data

## Overview
This folder contains machine learning metrics, evaluation results, and data summaries generated from the technical ML analysis.

---

## 📊 Files

### metrics_summary.json
Comprehensive JSON file with all ML evaluation metrics.

**Contents**:
- Baseline model metrics (train/test accuracy, precision, recall, F1, ROC-AUC)
- Final model metrics (all above + cross-validation scores)
- Confidence intervals for all test metrics
- Confusion matrices (baseline and final)
- Feature importance rankings
- Class distribution in test set

**Structure**:
```json
{
  "baseline_metrics": {...},
  "final_metrics": {
    "train_accuracy": 0.9996,
    "test_accuracy": 0.7601,
    "precision_usa": 0.7674,
    "recall_usa": 0.7665,
    "f1_usa": 0.7669,
    "roc_auc": 0.8376,
    "cv_mean": 0.7594,
    "cv_std": 0.0054,
    "accuracy_ci": [0.7458, 0.7744],
    ...
  },
  "confusion_matrix_baseline": [[...], [...]],
  "confusion_matrix_final": [[...], [...]],
  "class_distribution": {...}
}
```

**How to use**:
```python
import json
with open('metrics_summary.json', 'r') as f:
    metrics = json.load(f)

print(f"Test Accuracy: {metrics['final_metrics']['test_accuracy']:.4f}")
print(f"95% CI: {metrics['final_metrics']['accuracy_ci']}")
```

## 🔄 How to Generate These Files

The metrics are generated by a reproducible Python script:

```bash
cd ../../scripts
python3 generate_metrics.py
```

This will regenerate:
- `metrics_summary.json`
- `results_table.csv`
- Visualization files (in `../../assets/papers/`)

**Parameters** (fixed for reproducibility):
- Random state: 52
- Train/test split: 80/20
- Cross-validation: 5-fold stratified
- Model: RandomForestClassifier(criterion='entropy', max_depth=100)

## 📋 Data Dictionary

### Final Model Metrics

| Field | Type | Description |
|-------|------|-------------|
| `train_accuracy` | float | Accuracy on training set |
| `test_accuracy` | float | Accuracy on held-out test set |
| `precision_usa` | float | True positives / (TP + FP) for USA class |
| `recall_usa` | float | True positives / (TP + FN) for USA class |
| `f1_usa` | float | Harmonic mean of precision and recall |
| `roc_auc` | float | Area under ROC curve |
| `cv_mean` | float | Mean accuracy across 5 folds |
| `cv_std` | float | Standard deviation across folds |
| `accuracy_ci` | list | [lower, upper] bounds of 95% CI |
| `top_20_features` | list | Feature importance rankings |

## 💡 Interpreting the Metrics

**High Test Accuracy (76.01%)**
- Model correctly classifies 3 out of 4 recipes
- Well above random (50%) for balanced classes
- Improvement from baseline (54.14%) suggests ingredients are discriminative

**95% Confidence Interval [74.58%, 77.44%]**
- 95% confident true accuracy is in this range
- Narrow interval suggests stable, reproducible results
- Not likely due to random chance

**Consistent Cross-Validation (75.94% ± 0.54%)**
- Low std dev indicates consistent performance across folds
- Good generalization to unseen data
- No signs of hyperparameter overfitting

**High ROC-AUC (0.8376)**
- Model has strong discriminative ability
- For each random USA-EUR pair, model ranks USA higher 84% of time
- Substantial improvement from baseline (0.5596)
